# Saudi Investment Analyzer - Scheduled Netlify Version

هذه نسخة محدثة تعمل مع Netlify Functions.

- تحديث يومي الساعة 5:00 مساءً بتوقيت السعودية.
- 5 مساءً السعودية = 14:00 UTC.
- endpoint الأسعار: /.netlify/functions/prices
- اختبار فوري: /.netlify/functions/prices?refresh=1

مهم: انشر المشروع عبر GitHub + Netlify أو Netlify CLI حتى تعمل Functions و Scheduled Functions. السحب والإفلات كملف HTML فقط لا يكفي لهذه النسخة.
